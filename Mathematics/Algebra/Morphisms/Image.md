> [!tip] Definition
> Image describes that what you can reach using the map (i.e. how much of the map is covered). Strictly:
> $$
> \text{Im}(\phi) := \{y \in H \ | \ \exists x \in G, \ \phi(x) = y\}
> $$

If the image is the whole target the map is *surjective*. For example:
1. For $\phi(n) = 2n$ the kernel $Im(\phi) = 2\mathbb{Z}$
2. For $\phi: \mathbb{Z} \to \mathbb{Z} / 6\mathbb{Z}, \quad \phi(n) = n \text{ mod } 6$ the image is $\mathbb{Z}_{6}$ (meaning $\{0, 1, 2, 3, 4, 5\}$)
