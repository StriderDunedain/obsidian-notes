*Bolzano’s Nullstellensatz* is a specific case of [[Intermediate Value Theorem]] that says: If a function goes from positive to negative and vice versa, there has to be such a number that is a root of this function:
$$
f: [a, b] =
\begin{equation}
\begin{cases}
 f(a) < 0 \\
 f(b) > 0
\end{cases}
\implies c \in (a, b) \text{ where } f(c) = 0
\end{equation}
$$
**Note**:
- $a$ and $b$ must have different signs
- function $f$ must be continuous
