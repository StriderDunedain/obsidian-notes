Die Mächtigkeit, or Kardinalität, means the number of elements in the set and is denoted as $|M|$. Sets can be either **abzählbar** — elements can either be counted or can be mapped onto $\mathbb{N}$ — or **überabzählbar** which is too big to be counted. Also, very important is that they are bijective (since all elements of one set have to be mapped to exactly one element in another set, more on that in [[Mapping]]). Some special notations are:
$$
\begin{align}
\mathbb{N} \text{ has the cardinality of } \aleph_{0} \\
\mathbb{R} \text{ has the cardinality of } \mathfrak{c}
\end{align}
$$
$\mathbb{R}$'s cardinality is called *Continuum*, $\mathbb{N}$'s – aleph zero. The two of them prompted Cantor to raise the [[The Continuum Hypothesis]]
