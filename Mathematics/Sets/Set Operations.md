1. **Intersection / Durchschnitt**
$$M \cap N := \{x|x \in M \ \wedge \ x \in N\}$$
Two sets are **disjoint** when their intersection gives an empty set (i.e. they do not cross)

2. **Union / Vereinigung**
$$M \cup N := \{x|x \in M \ \vee \ x \in N\}$$
3. **Difference / Differenzmenge**
$$M \setminus N := \{x|x \in M \ \wedge \ x \notin N\}$$
Can be expanded as:
$$M \setminus N = M \cap \overline{N}$$
If the $N$ is a subset of $M$ then the $M \ N$ is also called a **complement** and is denoted as: $\overline{N}$
