### Definition and Properties
A set can:
1. Contain other sets (even the empty set[^1])
2. Every element of a set is unique


- $M$ **is a subset (Teilmenge)** of $N$ and $N$ **is a superset** of $M$:
$$ M \subset N \ ,\ N \supset M$$
- $M$ is **not a subset** of $N$:
$$M \not\subset N$$
- $M$ is a **proper subset (echte Teilmenge)** of $N$ ($M$ is a subset and $M \neq N$):
$$M \subsetneq N$$

A **power set (Potenzmenge)** is a set of all sets of a set:
$$P(M)$$
For example for the set $M := \{1, 2\}$ the power set is:
$$P(M) := \{\emptyset, \ \{1\}, \ \{2\}, \ \{1, 2\}\}$$

[^1]: An interesting explanation is the [[Vacuous Truth]]
