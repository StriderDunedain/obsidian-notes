A cartesian product is defined as follows and is a sort of combination of elements from $M$ as well as from $N$. Every pair is set in stone so that $(1, 5) \neq(5, 1)$
$$M \times N := \{(x, y)| x \in M, \ y \in N\}$$

The definition for a 2D plane for drawing functions on is defined as follows:
$$\mathbb{R}^2 := \mathbb{R} \times \mathbb{R} = \{(x, y) | \ x, y \in \mathbb{R}\}$$
A parable is formed as follows:
$$P := \{(x, x^2) | \ x\in \mathbb{R}\}$$
