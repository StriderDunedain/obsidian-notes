1. **Queries per Second** (QPS)
	How many requests are coming in
	$$QPS = Concurrency \ \div \ avg(RT)$$
2. **Transactions per Second** (TPS)
	How many requests are being completed
3. **Concurrency** 
	How many requests are being handled simultaneously
4. **Response Time** (RT)
	How long does a request take to and from a server back to the client
5. **Daily Active Users** (DAU)
		That is self-explanatory
