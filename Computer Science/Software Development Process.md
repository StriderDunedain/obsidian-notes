> [!abstract] Sequential model (Waterfall)
> You go through the development stages strictly one after the other and only once

Typical phases are:
- Requirements
- Design
- Implementation
- Testing
- Deployment

---
> [!abstract] Incremental model
> Build the system incrementally, in pieces, each adding a functionality

---
> [!abstract] Iterative model
> Build  -> test -> improve
